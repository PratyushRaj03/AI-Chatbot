# 🤖 AI Career Assistant Chatbot

A modern, responsive AI chatbot designed to provide career guidance, skill recommendations, and project suggestions for students and developers.

This project is being built step-by-step with a focus on **clean UI, real-world usability, and AI integration**.

---

## 🚀 Features (Current)

- 💬 Modern chatbot UI (inspired by ChatGPT/Gemini)
- 🎨 Clean and responsive design
- ⚡ Real-time message rendering
- 🤖 Simulated bot responses (frontend logic)
- ⏳ Typing indicator animation
- 🕒 Timestamped messages
- ⌨️ Enter key support for sending messages
- 📱 Mobile-friendly layout

---

## 🧠 Planned Features

- 🔗 Backend integration (FastAPI)
- 🤖 AI-powered responses using LLM APIs
- 🧠 Conversation memory (context-aware replies)
- 📄 Resume upload & analysis (PDF parsing)
- 🎯 Personalized career recommendations
- 💾 Chat history storage
- 🌐 Deployment (live app)
